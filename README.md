# Expense Tracker (Flask)

A personal expense tracking web app built with Flask and SQLite, featuring:
- Add, edit, and delete expenses
- Filter by category, payment method, user, and date range
- View summary totals

## Tech Stack
- Python & Flask
- SQLite (local dev DB)
- HTML, Jinja2, Bootstrap
